`deprecated` classifies *entries* that no longer describe the intended project direction.

A deprecated entry stays in the lake when the old design still explains current repository material,
migration context, or terms that appear in nearby entries.
The category tells readers to treat the entry as historical or transitional design.

Use `deprecated` before removing an entry when the old design still has review value.
A replacement entry should be linked through ordinary structure when that improves navigation.
